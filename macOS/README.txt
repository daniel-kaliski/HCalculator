Rozwiązywanie problemu "Aplikacja jest uszkodzona" na macOS
Jeśli podczas próby uruchomienia programu na komputerze Mac zobaczysz komunikat: "Aplikacja HCalculator.app jest uszkodzona i nie można jej otworzyć", nie martw się. Plik nie jest zepsuty. Jest to standardowe zabezpieczenie systemu macOS (Gatekeeper), które blokuje aplikacje pobrane z przeglądarki.
Aby jednorazowo odblokować aplikację, wykonaj te krótkie kroki:
	1.	Otwórz aplikację Terminal (naciśnij Cmd + Spacja, wpisz "Terminal" i wciśnij Enter).
	2.	W oknie Terminala wpisz poniższą komendę (pamiętaj o dodaniu spacji na samym końcu!):
xattr -cr 
	3.	Znajdź rozpakowaną aplikację HCalculator.app w systemie (np. w folderze Pobrane).
	4.	Przeciągnij ikonę aplikacji i upuść ją w oknie Terminala (ścieżka do pliku doda się automatycznie po spacji).
	5.	Wciśnij Enter. (W terminalu nie pojawi się żaden nowy komunikat, to normalne).
	6.	Uruchom aplikację HCalculator ponownie dwukrotnym kliknięciem.
🇬🇧 Angielski (English)
Fixing the "App is damaged" error on macOS
If you see the message: "HCalculator.app is damaged and can't be opened" when trying to run the program on a Mac, don't worry. The file is not actually broken. This is a standard macOS security feature (Gatekeeper) that quarantines apps downloaded via a web browser.
To unblock the app, follow these quick steps:
	1.	Open the Terminal application (press Cmd + Space, type "Terminal", and press Enter).
	2.	In the Terminal window, type the following command (make sure to add a space at the very end!):
xattr -cr 
	3.	Locate the extracted HCalculator.app in your Finder (e.g., in the Downloads folder).
	4.	Drag and drop the app icon into the Terminal window (the file path will automatically be added after the space).
	5.	Press Enter. (The terminal will not display any confirmation message, this is normal).
	6.	Double-click HCalculator to run it again.
