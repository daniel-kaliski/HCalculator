## Über das Programm

HCalculator ist eine kostenlose, leichte Desktop-Anwendung, die für Ingenieure, Mechaniker, Instandhaltungspersonal und Konstrukteure von Hydrauliksystemen entwickelt wurde. Das Programm arbeitet zu 100 % offline und ist damit das ideale Werkzeug für die Arbeit direkt in der Produktionshalle, in der Werkstatt oder im Feld, wo der Internetzugang oft eingeschränkt ist.

Mit HCalculator können Sie grundlegende Konstruktionsberechnungen schnell und präzise durchführen, ohne in Formeln und Tabellen blättern zu müssen.

## Hauptfunktionen (5 Werkzeuge in 1)

* **Zylinderkraft berechnen:** Schneller Rechner für Druck- und Zugkraft.
* **Zylindergeschwindigkeit:** Berechnung der Aus- und Einfahrzeit der Kolbenstange.
* **Pumpenleistung:** Auswahl und Berechnung des Durchflusses in Litern pro Minute.
* **Leistungsbedarf:** Berechnung der Motorleistung, die zum Antrieb einer Hydraulikpumpe bei einem bestimmten Druck erforderlich ist.
* **Offline-Betrieb:** Keine Verzögerungen, keine Werbung, volle Datenprivatsphäre.

## Installations- und Startanleitung

Das Programm HCalculator für Windows ist eine **portable** Anwendung. Das bedeutet, dass es nicht im System installiert wird und die Registrierung nicht verändert.

1. Laden Sie die Datei HCalculator.zip herunter.
2. **Wichtig:** Entpacken Sie den gesamten Inhalt des Archivs in einen neuen Ordner (z. B. auf den Desktop oder einen USB-Stick). Starten Sie das Programm nicht direkt aus der .zip-Dateivorschau!
3. Öffnen Sie den entpackten Ordner und doppelklicken Sie auf die Datei HCalculator.exe.

## Fehlerbehebung (Windows SmartScreen-Warnung)

Wenn Sie das Programm zum ersten Mal starten, erscheint möglicherweise eine blaue **Windows SmartScreen**-Schutzmeldung ("Der Computer wurde durch Windows geschützt"). Dies ist ein Standardverhalten des Systems bei neuen, unabhängigen Anwendungen.

**Um das Programm zu starten:**
1. Klicken Sie im Meldungsfenster auf den Text "Weitere Informationen".
2. Unten im Fenster erscheint eine neue Schaltfläche – klicken Sie auf "Trotzdem ausführen".
3. Bei jedem weiteren Start wird das Programm ohne diese Meldung ausgeführt.

## Sprachversionen

HCalculator unterstützt 4 Sprachen: Polnisch, Englisch, Deutsch und Rumänisch.
Das Programm versucht standardmäßig, sich an die Sprache Ihres Betriebssystems anzupassen. Sie können die Sprache auch manuell in der Anwendungsoberfläche ändern.
