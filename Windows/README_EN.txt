## About the program

HCalculator is a free, lightweight desktop application created for engineers, mechanics, maintenance staff, and hydraulic system designers. The program works 100% offline, making it an ideal tool for working directly on the production floor, in the workshop, or in the field where internet access may be limited.

With HCalculator, you can quickly and precisely perform basic engineering calculations without flipping through formulas and tables.

## Main features (5 tools in 1)

* **Cylinder force calculation:** Quick calculator for pushing and pulling force.
* **Cylinder speed:** Calculating extension and retraction time of the piston rod.
* **Pump flow rate:** Selection and calculation of flow in liters per minute.
* **Power requirement:** Calculating the motor power needed to drive a hydraulic pump at a given pressure.
* **Offline operation:** No delays, no ads, full data privacy.

## Installation and startup instructions

The HCalculator program for Windows is a **Portable** application. This means it does not install in the system and does not modify the registry.

1. Download the HCalculator.zip file.
2. **Important:** Extract the entire contents of the archive to a new folder (e.g., on your Desktop or a USB drive). Do not run the program directly from the .zip file preview!
3. Open the extracted folder and double-click the HCalculator.exe file.

## Troubleshooting (Windows SmartScreen Warning)

When you run the program for the first time, a blue **Windows SmartScreen** protection message may appear ("Windows protected your PC"). This is standard system behavior for new, independent applications.

**To run the program:**
1. In the message window, click the "More info" text.
2. A new button will appear at the bottom of the window - click "Run anyway".
3. The program will start without this message on every subsequent launch.

## Language versions

HCalculator supports 4 languages: Polish, English, German, and Romanian.
The program tries to adapt to your operating system language by default. You can also change the language manually in the application interface.
